Label

An accessible label element for form controls. Supports required indicators and pairs naturally with inputs, selects, and textareas.

Preview

This will be your public display name.

Code

<!-- Basic label with input -->
<div class="space-y-2">
  <label class="mw-label" for="email-input">Email address</label>
  <input type="email" id="email-input" class="mw-input" placeholder="you@example.com">
</div>

<!-- Label with required indicator -->
<div class="space-y-2">
  <label class="mw-label" for="name-input">
    Full name <span class="text-destructive">*</span>
  </label>
  <input type="text" id="name-input" class="mw-input" placeholder="Enter your full name">
</div>

<!-- Label with description text -->
<div class="space-y-2">
  <label class="mw-label" for="username-input">Username</label>
  <input type="text" id="username-input" class="mw-input" placeholder="marrow-user">
  <p class="text-xs text-muted-foreground">This will be your public display name.</p>
</div>

Disabled State

When paired with a disabled input, the label can be visually dimmed to indicate the field is inactive.

<div class="space-y-2">
  <label class="mw-label opacity-50" for="disabled-input">Disabled field</label>
  <input type="text" id="disabled-input" class="mw-input" placeholder="Cannot edit" disabled>
</div>
