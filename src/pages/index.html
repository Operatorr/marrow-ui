Why Marrow UI?

A different approach to UI kits. No React, no virtual DOM, no complex state management. Just HTML, CSS, and a sprinkle of Alpine.js.

No Framework Lock-in

Pure HTML and Tailwind CSS. Interactivity through Alpine.js. Works with any backend or static site generator.

Themeable

One config file controls everything: colors, radius, spacing, fonts, compactness. Light and dark mode out of the box.

60+ Components

From buttons and cards to data tables and command palettes. Everything you need to build complete applications.

Copy & Paste

Each component is self-contained. Copy the HTML, drop it in your project. No npm install, no imports, no bundler.

Compactness Control

Switch between minimal, compact, normal, relaxed, and spacious modes. Perfect for data-dense dashboards or spacious marketing sites.

Alpine.js Powered

All interactivity uses Alpine.js. Lightweight, declarative, and right in your HTML. No virtual DOM overhead.

Complete control.

Every aspect of your UI is controlled from marrow.config.js. Colors, radius, compactness, fonts, shadows, transitions. All documented with available options.

HSL colors for easy opacity control
5 compactness levels: minimal to spacious
Dual font system (heading + body)
Choose any Iconify icon library
Light & dark mode with full color control 
// marrow.config.js
module.exports = {
  theme: "light",
  colors: {
    light: {
      primary: "220 90% 56%",
      background: "0 0% 100%",
      // ... full color palette
    }
  },
  radius: "lg",        // none|sm|md|lg|xl|2xl|full
  compactness: "normal",// minimal|compact|normal|relaxed|spacious
  fonts: {
    heading: "Inter",
    body: "Inter",
  },
  menuAccent: "subtle", // subtle|bold|border
  iconLibrary: "lucide",
  shadow: "sm",         // none|sm|md|lg|xl
};
