Usage

Use .mw-separator for a horizontal divider that spans the full width with vertical margin. Use .mw-separator-vertical for a vertical divider inside a flex container — make sure the parent has a set height (e.g., h-5) so the line is visible.

No JavaScript is required. The separator color is controlled by the --marrow-border CSS variable.
