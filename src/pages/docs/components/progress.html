Progress

A progress bar that visually indicates completion status. Uses Alpine.js to manage the progress value reactively.

Preview

Progress

Code

<div x-data="mwProgress(60)" class="space-y-2">
  <div class="flex justify-between text-sm">
    <span class="text-muted-foreground">Progress</span>
    <span class="font-medium" x-text="value + '%'"></span>
  </div>
  <div class="mw-progress">
    <div class="mw-progress-bar" :style="'width: ' + value + '%'"></div>
  </div>
</div>

Sizes

Adjust the height of the progress bar using utility classes.

Small (h-1)
Default (h-2)
Large (h-4) 
<!-- Small -->
<div class="mw-progress" style="height: 0.25rem;">
  <div class="mw-progress-bar" style="width: 40%;"></div>
</div>

<!-- Default -->
<div class="mw-progress">
  <div class="mw-progress-bar" style="width: 60%;"></div>
</div>

<!-- Large -->
<div class="mw-progress" style="height: 1rem;">
  <div class="mw-progress-bar" style="width: 80%;"></div>
</div>

API

Property Type Description
value Number Current progress value (0 to 100)
set(v) Method Set the progress value, clamped between 0 and 100
