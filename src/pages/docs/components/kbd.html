Kbd

A keyboard shortcut indicator used to display individual keys or key combinations inline with text.

Preview

Single keys

K Enter Tab Esc Space

Key combinations

Ctrl + K Ctrl + Shift + P Cmd + S

Inline with text

Press Ctrl + K to open the command palette.

Code

<!-- Single key -->
<kbd class="mw-kbd">K</kbd>

<!-- Key combination -->
<span class="flex items-center gap-1">
  <kbd class="mw-kbd">Ctrl</kbd>
  <span class="text-xs text-muted-foreground">+</span>
  <kbd class="mw-kbd">K</kbd>
</span>

<!-- Three-key combination -->
<span class="flex items-center gap-1">
  <kbd class="mw-kbd">Ctrl</kbd>
  <span class="text-xs text-muted-foreground">+</span>
  <kbd class="mw-kbd">Shift</kbd>
  <span class="text-xs text-muted-foreground">+</span>
  <kbd class="mw-kbd">P</kbd>
</span>

<!-- Inline with text -->
<p>Press <kbd class="mw-kbd">Ctrl</kbd>
  <span class="text-xs text-muted-foreground">+</span>
  <kbd class="mw-kbd">K</kbd> to open the command palette.
</p>

In Menu Items

Keyboard shortcuts are commonly shown alongside menu items using a trailing kbd element.

Cut Ctrl+X
Copy Ctrl+C
Paste Ctrl+V 
<div class="mw-menu-item justify-between">
  <span>Copy</span>
  <span class="flex items-center gap-1">
    <kbd class="mw-kbd">Ctrl</kbd>
    <span class="text-xs text-muted-foreground">+</span>
    <kbd class="mw-kbd">C</kbd>
  </span>
</div>
