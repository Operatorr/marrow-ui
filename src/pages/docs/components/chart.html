Production Use

Use a charting library for production

The CSS-based chart above is great for simple visualizations. For production dashboards with interactivity, tooltips, and complex data, integrate with a dedicated charting library such as Chart.js, ApexCharts, or D3.js. These libraries work seamlessly alongside Marrow UI.

<!-- Example: Chart.js inside a Marrow card -->
<div class="mw-card">
  <h3 class="font-heading font-semibold mb-4">Analytics</h3>
  <canvas id="myChart"></canvas>
</div>

<script src="https://cdn.jsdelivr.net/npm/chart.js"></script>
<script>
  new Chart(document.getElementById('myChart'), {
    type: 'bar',
    data: {
      labels: ['Jan', 'Feb', 'Mar', 'Apr', 'May', 'Jun'],
      datasets: [{
        label: 'Revenue',
        data: [6500, 7800, 5200, 9000, 7200, 10000],
      }]
    }
  });
</script>
