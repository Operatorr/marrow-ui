Slider

A styled range input that lets users select a value within a defined range by dragging a thumb along a track.

Code

<div x-data="mwSlider(0, 100, 50)" class="space-y-4">
  <div class="flex items-center justify-between">
    <label class="mw-label">Volume</label>
    <span class="text-sm text-muted-foreground" x-text="value + '%'"></span>
  </div>

  <div class="relative pt-1">
    <!-- Visual track -->
    <div class="mw-slider-track">
      <div class="mw-slider-range" :style="'width: ' + percentage + '%'"></div>
    </div>
    <!-- Invisible native range input -->
    <input type="range" :min="min" :max="max"
           x-model="value" @input="handleInput($event)"
           class="absolute inset-0 w-full opacity-0 cursor-pointer"
           style="margin: 0; height: 0.375rem;">
  </div>
</div>

Usage

Initialize with x-data="mwSlider(min, max, initial)". The component exposes value (current number) and percentage (computed 0–100 scale for the fill bar).

The visual track uses .mw-slider-track and .mw-slider-range for styling. A transparent native <input type="range"> is layered on top for accessibility and keyboard support.
